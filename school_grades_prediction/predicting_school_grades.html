Predicting School Grades¶

Problem¶

We want to use Scikit's Support Vector Regression (SVR) to predict final school grades of high school students based on a number of features (school, demographics and social) of each student.

Given¶

This file traindata.csv contains the data of the students. The first row is the header. There are 549 students in the training set and each student is represented by 30 attributes. The last column contains the grade of the student which is to be predicted.

To-Do¶

We are to apply SVR to this data and obtain a regression model to predict the grade based on the 30 parameters of each student. We can do model selection (type of kernel, kernel parameters, epsilon, C, etc.) through cross-validation on this data. We can either try to minimize the root mean square error or maximize the pearson or spearman correlation coefficients in our model selection.

Testing¶

Once we have trained your model, our model will be tested on the test file: testfeatures.csv. The test file contains 100 test examples and has the same format as train.csv except that the last column (GRADES) is not present. We are to predict the GRADE of each example in the test file.
